## Bootstrap starter

Include [Bootstrap](https://getbootstrap.com) CSS and JS via the [jsDelivr CDN](https://www.jsdelivr.com/package/npm/bootstrap).

## Download

```sh
git clone https://github.com/twbs/examples.git
cd examples/starter/
open index.html
```

## Testing locally

```sh
npm install
npm start
```
